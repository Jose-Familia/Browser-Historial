# Browser-Historial

¡Bienvenido a **Browser-Historial**!  
Este proyecto tiene como objetivo gestionar, analizar y visualizar el historial de navegación web de los usuarios de manera eficiente y segura.

## Descripción

**Browser-Historial** es una aplicación diseñada para importar, organizar y estudiar el historial de diferentes navegadores web. Puede servir para análisis personales, fines educativos o tareas de investigación sobre hábitos de navegación. El proyecto prioriza la privacidad de los datos y la facilidad de uso.

## Características principales

- **Importación de historial**: Soporta múltiples navegadores populares.
- **Visualización amigable**: Consulta tu historial filtrando por fechas, dominios o términos clave.
- **Análisis estadístico**: Obtén información sobre hábitos, páginas más visitadas y patrones de navegación.
- **Exportación**: Descarga tu historial procesado en varios formatos.
- **Privacidad**: Todo el procesamiento se realiza localmente. Nadie más tiene acceso a tu información.
